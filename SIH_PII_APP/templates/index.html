How to Use the PII Redactor

Follow these steps to redact PII from your PDF:

  • Browse or Drop File: Click the "BROWSE FILE" button to select a file, or drag and drop a file into the drop zone.
  • Check PII: Click the "PROCESS FILE" button to start the processing.
  • Wait for Processing: The application will process the file. Please wait for the operation to complete.
  • After Processing: The file will open on a new tab, and it will also be stored locally.
